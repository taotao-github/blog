# 2. 修改mongoDB数据库配置文件

找到mongoDB数据库的配置文件：C:/ProgramFiles/MongoDB/Server/4.0/bin/mongod.cfg（默认安装位置）。然后按照如下配置：

security：
     authorization:enabled # 注意缩进，参照其他的值来改，若是缩进不对可能导致后面服务不能重启
1
2

# 3. 重启mongoDB服务

配置完成后，重启服务：找到服务，右击重启服务。

**原因：**配置时缩进不对。enabled前面有一个空格

security：
     authorization: enabled # 注意缩进，参照其他的值来改，若是缩进不对可能导致后面服务不能重启
1
2

修改后就能成功启动服务了。
